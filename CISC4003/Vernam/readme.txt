Use test.html to enter the panel. Must have JavaScript enabled in your browser.

1. En with a random key: input texts in the upper column, and leave the lower "key" column empty, then click "Encrypt:"
2. De with a random key: [input same with 1.], then click "Decrypt:"
3. En with a user input key: input texts in the upper column, and input keys in the lower "key" column empty, then click "Encrypt:"
4. De with a user input key: [input same with 3.], then click "Decrypts:"

5. Program exit: click "Leave (halt and close program): " and click "ok" in next pop up.
